# 地球刮刮乐

一个H5小游戏，基于Three.js、TypeScript、Vue。
<br />
选择正确的城市名，即可一层一层刮开地球。完全刮开地球即可胜利。
<br />
<br />

## demo

 <a>https://zao-lab.com/#/di-qiu-gua-gua-le</a>
<br />
<br />

## 项目依赖安装
```
yarn install
```
<br />

## 启动项目
```
yarn serve
```
